About the Palermo Scale

The Palermo Scale measures risk relative to background impact frequency. A value of −2 means the object is 100× less likely to hit than a random similarly-sized object over the same period. Everything currently tracked is well below zero.

☄️ NEO Close Approach Tracker

A live mission-control dashboard tracking near-Earth objects using real-time data from NASA and JPL planetary defense systems.

Data Sources:

  • NASA NeoWs API — weekly close approaches
  • JPL SBDB Close-Approach API — historical & future data
  • JPL Sentry Impact Monitoring — collision risk assessment

Built with globe.gl for 3D visualization. No server required — all data fetched live from public APIs.
